# Ecosomier Mendoza – Agile Ecommerce

Proyecto de organización y planificación ágil para el emprendimiento **Ecosomier Mendoza**, orientado a la venta de sommiers, colchones y muebles.

## 🛏️ Sobre el proyecto

Este repositorio documenta el proceso de análisis, planificación y organización de un ecommerce utilizando metodologías ágiles.

El objetivo es estructurar el negocio digital desde la experiencia real de un emprendimiento local.

## 🎯 Objetivos

- Definir el alcance del ecommerce
- Identificar roles y responsabilidades
- Planificar funcionalidades clave
- Organizar tareas mediante enfoque ágil

## 🧰 Herramientas utilizadas

- GitHub
- Metodologías ágiles
- Documentación colaborativa
